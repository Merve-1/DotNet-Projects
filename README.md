# Movie Ticket Booking System 

This assignment extends the Movie Ticket Booking System by introducing **Object-Oriented Programming concepts in C#**, including **abstraction, abstract classes, partial classes, and extension methods**.
The goal is to improve the system design, enforce stronger type rules, and add reusable utilities without modifying existing classes.

---

# 📚 Concepts Covered
* Abstract Classes
* Abstract Methods
* Virtual Methods
* Polymorphism
* Partial Classes
* Extension Methods
* Clean Code Organization

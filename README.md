### Data Structures in C# (Collections Framework)
| Structure                  | Real-world idea                 |
| -------------------------- | ------------------------------- |
| `Dictionary<TKey, TValue>` | Key → Value lookup (like a map) |
| `SortedDictionary`         | Automatically sorted map        |
| `HashSet<T>`               | Unique items only               |
| `Queue<T>`                 | FIFO (like a line)              |
| `Stack<T>`                 | LIFO (like undo)                |

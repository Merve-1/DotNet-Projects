### Data Structures in C# (Collections Framework)
| Structure                  | Type           | Real-world idea                 |
| -------------------------- | -------------- | ------------------------------- |
| `Dictionary<TKey, TValue>` | Data structure | Key → Value lookup (like a map) |
| `SortedDictionary`         | Data structure | Automatically sorted map        |
| `HashSet<T>`               | Data structure | Unique items only               |
| `Queue<T>`                 | Data structure | FIFO (like a line)              |
| `Stack<T>`                 | Data structure | LIFO (like undo)                |
